# ONX - Neural Empire Manager

🚀 **Otonom YouTube İmparatorluk Yönetim Sistemi**

ONX, yapay zeka destekli, tam otomatik video üretim ve YouTube kanalı yönetim platformudur. Rakiplerinizi analiz eder, viral içerikler üretir ve kanalınızı otomatik olarak büyütür.

🚀 **ONX Ultimate Dev++ - Autonomous YouTube Empire Management System powered by AI. Next.js 14 + FastAPI + Gemini 2.0 Pro + Celery for complete content automation.**

## ✨ God Mode Özellikler

### 🧠 AI Beyin (Gemini 2.0 Pro)

- **Akıllı Senaryo Üretimi**: Viral potansiyeli yüksek senaryolar otomatik olarak üretilir
- **SEO Optimizasyonu**: Başlık, açıklama ve etiketler için otomatik SEO optimizasyonu
- **Persona Simülasyonu**: 3-5 farklı kullanıcı personasıyla otomatik yorum yanıtları
- **Thumbnail Analizi**: AI destekli thumbnail performans tahmini
- **Otomatik Video Üretim**: Konseptten yayına kadar tam otomatik üretim hattı
- **Hormozi Stili Altyazı**: Dinamik, çok renkli, dikkat çekici altyazılar
- **Akıllı Kırpma**: 16:9 videoları otomatik olarak 9:16 (Shorts) formatına çevir
- **Sidechain Ducking**: Arka plan müziği ile konuşma miksleme

### 🕵️ Rakip Casusluğu (The Reaper)

- **yt-dlp Entegrasyonu**: Rakip kanalları detaylı analiz et
- **VPH Trend Analizi**: View-per-Hour trendlerini takip et
- **İçerik Boşluğu Tespiti**: Rakiplerin kaçırdığı fırsatları belirle
- **Thumbnail Analizi**: Rakip thumbnail stratejilerini analiz et

### 🛡️ Grey-Hat Stratejiler

- **Algoritma Kalkanı**: YouTube "Tekrarlayan İçerik" filtresini baypas et
  - %0.1 rastgele kare hızı titremesi
  - Görünmez piksel gürültüsü
  - %1.01 mikro hız değişimi
- **Hayalet Etkileşim**: Otomatik ilk yorum ve persona yanıtları
- **Thumbnail A/B Testi**: Otomatik thumbnail testi ve optimizasyonu

### 📊 Türkçe Komuta Merkezi

- **Misyon Kontrolü**: Çok kanallı dashboard ile tüm kanalları yönet
- **Canlı Analiz**: Gerçek zamanlı sistem performansı ve YouTube Analytics
- **Keşfet Ekranı**: Trend konuları ve "Tek Tıkla Üret" özelliği
- **30 Günlük Takvim**: Sürükle-bırak içerik planlama

## 🏗️ Teknik Mimari

### AI & Media

- **edge-tts**: SSML destekli gerçekçi sesler

## 🚀 Kurulum

### Otomatik Kurulum

```bash
# install.bat dosyasını çalıştır
.\install.bat
```

### Manuel Kurulum

```bash
# Backend kurulumu
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

# Frontend kurulum
cd ../frontend
npm install

# Redis başlat
redis-server

# Backend'i başlat
cd ../backend
venv\Scripts\activate
uvicorn app.main:app --reload

# Frontend'i başlat
cd ../frontend
npm run dev
```

## 🎯 Kullanım

### 1. Kanal Oluştur

1. Dashboard'a gidin
2. "Yeni Kanal" butonuna tıklayın
3. Kanal bilgilerini ve nişi seçin
4. YouTube API anahtarlarını yapılandırın

### 2. Rakip Analizi

1. "Rakip Casusu" bölümüne gidin
2. Rakip kanal URL'lerini ekleyin
3. Otomatik analiz başlatın
4. İçerik boşluklarını ve fırsatları görüntüleyin

### 3. İçerik Üretimi

1. "AI Senaryo" bölümüne gidin
2. Konu ve hedef kitleyi belirtin
3. Senaryo oluştur butonuna tıklayın
4. Medya assets'lerini seçin
5. Video üretimini başlatın

### 4. Otomatik Yükleme

1. "Yükleme Ayarları"nı yapılandırın
2. Takvimde yayın zamanlarını planlayın
3. Otomatik yükleme özelliğini aktif edin
4. Sistem otomatik olarak videoları yüklesin
